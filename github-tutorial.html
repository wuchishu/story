如何在GitHub上發佈檔案（新手完整教學）

  • 建立帳號與Repository
  • 安裝Git工具
  • 上傳檔案到GitHub
  • 確認檔案
  • 常見問題

步驟一：建立GitHub帳號與Repository

進入GitHub官網，點擊「Sign up」註冊帳號；登入後點選「New repository」，設定名稱（如story）、描述與公開性（Public）。

步驟二：安裝與設定Git

從官方網站下載Git並安裝，透過終端機設定姓名及email，例如：

git config --global user.name "wuchishu"
git config --global user.email "您的email@example.com"

步驟三：將檔案上傳到GitHub（完整指令與實例）

  1. 複製Repository網址，例如https://github.com/wuchishu/story.git，並於終端機執行：
    git clone https://github.com/wuchishu/story.git
  2. 假設你在桌面創建一個名為github_tutorial的資料夾，內含一個檔案index.html，將此檔案複製到剛剛下載的資料夾（例如：story）中，然後進入該資料夾執行：
    git add .
  3. 紀錄此次更新訊息，例如：
    git commit -m "新增GitHub教學網頁"
  4. 將檔案推送至GitHub：
    git push origin main

步驟四：確認檔案是否上傳成功

登入GitHub確認檔案已經顯示在Repository。您可以訪問 https://github.com/wuchishu/story 來查看。

常見問題處理
